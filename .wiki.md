# Project Summary
The project is a pet-focused platform that fosters a community for pet lovers, providing user authentication, social interactions, and a marketplace for pet-related products and services. Users can connect, share experiences, and access high-quality pet products. Recent enhancements include a seller review system, product and shop rating systems, logistics management, and improved user profile management, enriching the user experience.

# Project Module Description
The project consists of two main modules:
- **Backend**: Manages user authentication, data management, and business logic, including product and shop management, as well as new features like seller reviews and ratings.
- **Frontend**: Provides the user interface for interaction with the platform, including social features, shopping experiences, and enhanced user profiles.

# File Description Inventory
- **Backend**: 
  - `main.py`: Entry point for the backend application.
  - `core/`: Core functionalities including authentication and database management.
  - `routers/`: Defines API endpoints for various features, including user management, product and shop ratings, logistics, and seller reviews.
  - `services/`: Contains business logic for handling data operations.
  
- **Frontend**: 
  - `src/App.tsx`: Main application component with routing.
  - `src/pages/*`: Various pages including user profiles, shops, and inspector dashboards.
  - `src/components/`: Reusable UI components.
  - `public/`: Static assets like images and favicon.
  - `dist/`: Contains built assets for the frontend.

# Technology Stack
- **Backend**: Python, FastAPI, SQLAlchemy, Alembic
- **Frontend**: React, TypeScript, Vite, Tailwind CSS, pnpm

# Usage
1. Install dependencies:
   ```
   cd app/frontend
   pnpm install
   ```
2. Build the frontend:
   ```
   pnpm run build
   ```
3. Run the backend:
   ```
   cd app/backend
   uvicorn main:app --reload
   ```
