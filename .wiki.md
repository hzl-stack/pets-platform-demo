# Project Summary
The project is a pet-focused platform that provides user authentication, social interactions with anonymity, and a marketplace for pet-related products and services. It aims to foster a community for pet lovers, allowing them to connect, share experiences, and access high-quality pet products.

# Project Module Description
The project consists of two main modules:
- **Backend**: Manages user authentication, data management, and business logic.
- **Frontend**: Offers the user interface for interaction with the platform, including social features and shopping experiences.

# Directory Tree
```
app/
├── backend/                 # Backend application files
│   ├── core/                # Core functionalities (auth, config, database)
│   ├── data_models/         # JSON data models for cart items, comments, posts, products
│   ├── dependencies/        # Dependency management
│   ├── lambda_handler.py    # AWS Lambda handler
│   ├── main.py              # Main entry point for the backend
│   ├── routers/             # API route definitions
│   └── services/            # Business logic services
└── frontend/                # Frontend application files
    ├── src/                 # Source files for the frontend
    ├── public/              # Public assets (images, favicon)
    ├── seo-scripts/         # Scripts for SEO purposes
    └── template_config.json  # Configuration for templating
```

# File Description Inventory
- **Backend**: 
  - `main.py`: Entry point for the backend application.
  - `core/`: Core functionalities including authentication and database management.
  - `routers/`: Defines API endpoints for various features.
  - `services/`: Contains business logic for handling data operations.
  
- **Frontend**: 
  - `src/App.tsx`: Main application component.
  - `src/pages/`: Contains different pages (Home, Social, Shop, Cart, Seller).
  - `src/components/`: Reusable UI components.
  - `public/`: Static assets like images and favicon.

# Technology Stack
- **Backend**: Python, FastAPI, SQLAlchemy, Alembic
- **Frontend**: React, TypeScript, Vite, Tailwind CSS, pnpm

# Usage
1. Install dependencies:
   ```
   cd app/frontend
   pnpm install
   ```
2. Build the frontend:
   ```
   pnpm run build
   ```
3. Run the backend:
   ```
   cd app/backend
   uvicorn main:app --reload
   ```
